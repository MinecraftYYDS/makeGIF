POST /api/encode

以 multipart/form-data 格式上传图片及参数，返回二进制动图文件。

表单字段

images # File[] — PNG/JPG，必填，最多 60 张 format # string — "gif" | "apng"，默认 "gif" fps # int — GIF 1~60 / APNG 1~120，默认 10 loop # int — 循环次数，0=无限，默认 0 width # int — 输出宽度 1~4096，需与 height 同时指定 height # int — 输出高度 1~4096，需与 width 同时指定

cURL 示例

curl -X POST https://your-worker.workers.dev/api/encode \ -F "images=@frame1.png" \ -F "images=@frame2.png" \ -F "fps=24" \ -F "format=apng" \ -o output.png
